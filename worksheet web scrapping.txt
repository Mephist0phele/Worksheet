Q What is the main difference between a web scraper and a web crawler?

answer :  main difference is that web crawlers usually focus on indexing the web while web scrapers extract or "scrape" data from webpages.

Q What is ‘robots.txt’ file? What is the use of ‘robots.txt’ file?

answer : A robots.txt file is a set of instructions for bots. This file is included in the source files of most websites. 
Robots.txt files are mostly intended for managing the activities of good bots like web crawlers, since bad bots aren't likely to follow the instructions.
